  club.html         → Be Pro FC & Be Pro Elite teams, NISA costs, gallery
  training.html     → Online training programs + 3 pricing tiers
  merch.html        → Store (6 products)
  contact.html      → Contact form + FAQ
  css/

    i18n.js         → EN/ES bilingual toggle (all translations)

HOW TO MAKE CHANGES
-------------------
1. Open any .html file in a text editor to edit content
2. Edit css/style.css to change colors, fonts, spacing
3. Edit js/i18n.js to update translations (both EN and ES sections)
4. After changes, run in Terminal:
     cd /Users/franklopez/bepro-website && vercel deploy --prod
5. Hard refresh browser/phone to see updates
